Enigma is simple programme with timer, scramble, scramble-picture and displaying your 5 last attempts. Also enigma count your average time of 5 and 12 turns.
Enigma is written on python with framework PyQt5. Also in the folder there is a file 'scram_gen', which contains the scramble generator itself. You can use this file to generate scrambles
simply by indicating the name of cube and number of movements.
https://github.com/alvvaysxxx/pyscramblesgenerator - repository with scramble generator.
